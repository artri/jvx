This example is a pre-configured Eclipse Project.

Simply extract the zip archive and "Import an existing Project" with Eclipse.
Don't select the checkbox, to copy the project into your workspace!

Before you start the included launcher (JVx EE Example), start the database!
The database start script is available in the db directory. Be sure that you
cd to the db directory before starting the script (startDB.sh or startDB.bat).

Login with username and password: admin

The application is a standard JVx application.
